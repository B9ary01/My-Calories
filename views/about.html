About Page



This is a Calorie Buddy Dynamic Web Application based on Nodejs, Express and MongoDB. You can see more information about the application below:

  • Home page displays all links to other pages.
  • Users can register using register page.
  • User can login using login page.
  • WeatherForm helps to check the weather of different cities around the world.
  • API page displays available food registered in the database in JSON format.
  • Users can add food to the database using AddFood page.
  • Users can delete food using Foodlist page.
  • Users can update food using Foodlist page.
  • Users can search food using SearchFood page.
  • Users can logout using Logout page.
